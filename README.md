# Nike Sales Performance Analysis

## Project Overview
This project analyzes Nike sales data to identify trends, top-performing products, and revenue-driving regions.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- Identified top revenue-generating regions
- Analyzed monthly sales trends
- Determined top 10 products by revenue
- Performed correlation analysis

## Project Outcome
Generated actionable business insights to support decision-making.
